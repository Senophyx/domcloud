Catatan untuk Hosting Cloud dengan paket Gratis:

  1. Hosting Cloud Gratis cocok digunakan sebagai website pribadi, media belajar, pengembangan aplikasi, atau untuk menyiapkan website sebelum disebar ke publik.
  2. Hosting Cloud gratis memerlukan anda untuk aktif memperpanjang hosting setiap 3 bulan sekali.
  3. Paket Gratis difasilitasi dengan domain gratis melalui subdomain .dom.my.id.
  4. Domain gratis melalui subdomain .dom.my.id mendukung SSL Gratis melalui Let's Encrypt dan dapat diindeks dan didaftarkan ke Google Search Console.
  5. Paket Gratis juga termasuk Akses FTP dan Remote MySQL Database
  6. Paket Gratis tidak memfasilitasi fitur premium seperti Custom Domain, DNS Server, Regular Backup, AWStats Reporting dan Akses SSH

Catatan untuk Semua Paket Hosting Cloud Berbayar:

  1. Pembayaran diposes oleh iPaymu, yang mendukung
    • Virtual Account BNI dan Mandiri
    • Indomart dan Alfamart
    • QRIS (OVO, ShopeePay, GoPay, DANA, LinkAja, BCA Mobile)
  2. Dikenakan biaya transaksi Rp. 5.000 setiap kali melakukan transaksi.
    Anda dapat membeli hosting dalam beberapa tahun sekaligus beserta domainnya agar lebih murah.
  3. Penjelasan Singkat Fitur Premium
    • Custom Domain adalah penggunaan hosting untuk domain diluar domain gratis .dom.my.id
    • Subserver adalah akun tambahan yang dapat digunakan sebagai tempat subdomain.
      Jumlah subserver mencerminkan jumlah subdomain yang anda bisa pakai.
    • Gratis SSL menggunakan Layanan Let's Encrypt untuk mendaftarkan SSL untuk keperluan HTTPS
    • DNS Server digunakan agar DNS pada domain registrar dapat dikelola otomatis oleh Hosting melalui DNS Nameserver
    • Regular Backup digunakan agar anda dapat mengatur backup dalam hosting secara terjadwal
    • AWStats Reporting digunakan untuk melihat analitik (jumlah view) dalam website melalui akses log
    • Akses SSH digunakan untuk mengakses Console dalam hosting untuk keperluan eksesuki program seperti Git, Ruby dan Python
  4. Bandwidth yang dihitung adalah penggunaan data masuk dan keluar dalam interval 30 hari.
  5. Hosting ini tidak menyediakan email, namun anda dapat menggunakan servis lain (SendGrid) untuk mengirim dan menerima email.
