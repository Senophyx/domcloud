Storage adalah batas penyimpanan yang dihitung sebagai jumlah kapasitas file, database, dan data server lainnya. Sedangkan Bandwidth adalah batasan data yang dikirim dan diterima dalam tempo waktu tertentu.

Bandwidth yang tertera adalah bandwidth dalam hitungan per tahun. Namun pembatasan efektif dilakukan per bulan. Misalkan, untuk paket Freedom anda mendapatkan 6 GB bandwidth, maka anda dibatasi 500 MB tiap bulan. Pembatasan ini akan direset ulang tiap awal bulan.

Anda dapat melakukan penghitungan cepat dengan rata-rata ukuran website yakni 100 KB per halaman. Contoh, dengan pembatasan paket Freedom 500 MB per bulan, anda dapat trafic maksimum sekitar 5000 view per bulan. Angka ini dapat ditingkatkan lebih jauh apabila anda mengoptimasi ukuran website atau menggunakan CDN. Baca Lebih Lanjut Disini.

Beberapa hosting provider lain mengklaim menyediakan storage dan bandwidth Unlimited. Hal ini tidak selalu benar, karena hosting tersebut menggunakan "Penggunaan Wajar" / "Fair Usage" yang kita tidak tahu batasan "wajar" secara tepatnya. DOM Cloud mematahkan kerancuan ini dengan menjelaskan batasan yang benar-benar anda dapatkan disetiap rentang harganya.

DOM Cloud tidak membangun server sendiri. DOM Cloud merupakan layanan hosting diatas Insfrastruktur Microsoft Azure dan saat ini kami memasang server utama di Singapura.

Server DOM Cloud digaransi oleh Standard SSD Azure Cloud SLA, namun hal ini sendiri tidak menjamin downtime dikarenakan kesalahan operator atau serangan internet lainnya. Baca Syarat Layanan Kami untuk keterangan lebih lanjut.
