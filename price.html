Catatan untuk Semua Paket Hosting Cloud:

  1. Kuota yang tertulis adalah kuota kapasitas disk untuk seluruh elemen termasuk file, database, email dan semacamnya
  2. Bandwidth terhitung data masuk dan keluar secara akumulatif dan tersetel ulang otomatis setiap 30 hari
  3. Kami menghitungan dengan ukuran 1 KB = 1024 bytes
  4. Kelas premium mendapatkan kesempatan untuk mendaftarkan SSL Let's Encrypt, sedangkan paket lain dapat menggunakan Cloudflare sebagai pengganti.

Catatan untuk Hosting Cloud dengan paket Gratis:

  • Hosting Cloud gratis memerlukan anda untuk aktif memperpanjang hosting setiap 6 bulan sekali
  • Kami tidak memfasilitasi fitur tambahan seperti FTP, Remote Database, Custom Domain dan Email
