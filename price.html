Catatan untuk Hosting Cloud dengan paket Gratis:

  1. Hosting Cloud gratis memerlukan anda untuk aktif memperpanjang hosting setiap 6 bulan sekali
  2. Paket Gratis difasilitasi dengan domain gratis melalui subdomain svXX.my.dom.id dan sudah terpasang SSL wilcard dari domain induk.
  3. Domain gratis melalui subdomain svXX.my.dom.id tidak dapat terindeks oleh Search Engine dan tidak bisa didaftarkan untuk dukungan proteksi dari Cloudflare.
  4. Kami tidak memfasilitasi fitur premium untuk paket gratis seperti Remote Access, SSL Let's Encrypt, Subdomain, Akun Email, SpamAssasin dan Webalizer

Catatan untuk Semua Paket Hosting Cloud Berbayar:

  1. Dikenakan biaya transaksi Rp. 5000 setiap kali pembayaran, agar lebih hemat anda bisa membayar sekaligus hingga 5 tahun kedepan.
  2. Kuota yang tertulis adalah kuota kapasitas disk untuk seluruh elemen termasuk file, database, email, logging dan semacamnya
  3. Bandwidth terhitung data masuk dan keluar secara akumulatif dan tersetel ulang otomatis setiap 30 hari
  4. Kami menghitung dengan ukuran 1 KB = 1024 bytes
  5. Dukungan SSL meliputi Pendaftaran SSL Let's Encrypt secara otomatis
  6. Remote Access meliputi Akses FTP dan MySQL Remote Access
  7. Demi Keamanan, anda harus menggunakan custom domain sebelum dapat menggunakan fitur premium seperti Remote Access, SSL, Subdomain, Akun Email, SpamAssasin dan Webalizer.
